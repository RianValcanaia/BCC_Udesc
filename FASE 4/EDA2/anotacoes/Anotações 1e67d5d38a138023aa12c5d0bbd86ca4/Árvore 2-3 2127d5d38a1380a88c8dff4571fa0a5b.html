Árvore 2-3

A árvore 2-3 é uma estrutura de dados auto-balanceada utilizada para armazenar dados ordenados, oferecendo suporte eficiente para:

  • Acesso sequencial
  • Acesso aleatório (pesquisa)

Complexidade de Tempo

  • Acesso aleatório: O(log n)
  • Inserção: O(log n)
  • Remoção: O(log n)

Essa estrutura foi idealizada por J. E. Hopcroft em 1970, destacando-se por sua ótima isometria: cada subárvore à esquerda, centro ou direita contém aproximadamente a mesma quantidade de dados.

Características

  • Cada nó interno pode conter uma ou duas chaves.
  • Todas as folhas estão no mesmo nível.
  • Quando um nó tem uma única chave, ele possui duas subárvores:
    • Subárvore à esquerda contém valores menores que a chave.
    • Subárvore à direita contém valores maiores que a chave.
  • Quando um nó possui duas chaves, ele terá três subárvores:
    • Subárvore à esquerda: valores menores que a menor chave.
    • Subárvore central: valores entre as duas chaves.
    • Subárvore à direita: valores maiores que a maior chave.
  • É considerada uma árvore multi-vias, pois não é limitada a dois filhos como uma árvore binária.

Propriedades Importantes

  • Todas as folhas estão no mesmo nível.
  • A árvore cresce em direção à raiz, ao contrário de estruturas como ABB, AVL ou Rubro-Negra, que crescem pelas folhas.
  • Se um nó possui duas chaves, elas estão em ordem crescente.

Operações Fundamentais

  • Inserção de chave
  • Remoção de chave
  • Busca de chave
  • Percurso da árvore

Estrutura da árvore

Cada nó pode conter uma ou duas chaves, e até três ponteiros para subárvores (esquerda, centro e direita).

typedef struct _no {
    int num_chaves;       // Número de chaves no nó (1 ou 2)
    int chave_e, chave_d; // Chaves armazenadas
    struct _no *esq;      // Ponteiro para subárvore esquerda
    struct _no *centro;   // Ponteiro para subárvore central
    struct _no *dir;      // Ponteiro para subárvore direita
} No;

Criação de um novo nó

Cria e retorna um novo nó com as chaves e ponteiros fornecidos.

No *criaNo(int ch1, int ch2, int num_chaves, No *pe, No *pc, No *pd) {
    No *novo = (No *)malloc(sizeof(No));
    if (novo == NULL) {
        printf("Erro ao alocar memória\n");
        return NULL;
    }
    novo->chave_e = ch1;
    novo->chave_d = ch2;
    novo->num_chaves = num_chaves;
    novo->esq = pe;
    novo->centro = pc;
    novo->dir = pd;
    return novo;
}

Verificação de folha

Verifica se o nó atual é uma folha (ou seja, não possui filhos).

int ehFolha(No *no) {
    return (no->esq == NULL && no->centro == NULL && no->dir == NULL);
}

Adiciona uma chave e subárvore em um nó com uma única chave

Esta função adiciona uma nova chave a um nó que possui apenas uma chave. Se necessário, reorganiza os ponteiros para preservar a ordem.

void adicionaChave(No *no, int ch, No *p) {
    if (no == NULL || no->num_chaves != 1) {
        printf("Erro: nó nulo ou com número de chaves inválido\n");
        return;
    }

    if (ch < no->chave_e) {
        // Nova chave vai antes da chave existente
        no->chave_d = no->chave_e;
        no->chave_e = ch;
        no->dir = no->centro;
        no->centro = p;
    } else {
        // Nova chave vai depois da chave existente
        no->chave_d = ch;
        no->dir = p;
    }
    no->num_chaves = 2;
}

Localiza um nó a partir de uma chave

Busca recursivamente um nó que contenha a chave. Retorna o ponteiro para o nó se encontrado; caso contrário, retorna NULL.

No* localizaNo(No* no, int chave) {
    if (no == NULL)
        return NULL; // Não encontrou

    if (chave == no->chave_e)
        return no; // É a chave esquerda

    if ((no->num_chaves == 2) && (chave == no->chave_d))
        return no; // É a chave direita

    if (chave < no->chave_e)
        return localizaNo(no->esq, chave);

    else if ((no->num_chaves == 1) || (chave < no->chave_d))
        return localizaNo(no->centro, chave);

    else
        return localizaNo(no->dir, chave);
}

Quebra de nó (split) ao inserir em um nó com duas chaves

Quando um nó 2-3 já possui duas chaves e uma nova chave precisa ser inserida, ocorre uma quebra de nó (split). Essa operação divide o nó atual em dois nós menores e promove uma das chaves ao nível superior da árvore.

A função abaixo realiza essa quebra, reorganizando os ponteiros das subárvores adequadamente.

No *quebraNo(No *no, int valor, int *rval, No *subarvore) {
    No *aux;

    if (valor > no->chave_d) {
        // Novo valor é maior que todas as chaves
        *rval = no->chave_d; // Promove a maior chave
        aux = no->dir;
        no->dir = NULL;
        no->num_chaves = 1;
        return criaNo(valor, 0, 1, aux, subarvore, NULL);

    } else if (valor >= no->chave_e) {
        // Novo valor está entre as duas chaves
        *rval = valor; // Promove o novo valor
        aux = no->dir;
        no->dir = NULL;
        no->num_chaves = 1;
        return criaNo(no->chave_d, 0, 1, subarvore, aux, NULL);

    } else {
        // Novo valor é menor que todas as chaves
        *rval = no->chave_e; // Promove a menor chave
        aux = criaNo(no->chave_d, 0, 1, no->centro, no->dir, NULL);
        no->chave_e = valor;
        no->centro = subarvore;
        no->dir = NULL;
        no->num_chaves = 1;
        return aux;
    }
}

Inserção em Árvore 2-3

A inserção em uma árvore 2-3 segue regras que garantem que a estrutura permaneça balanceada, com todas as folhas no mesmo nível.

Regras de Inserção

  • Regra 1: Árvore vazia
    • Cria um novo nó e insere o valor como chave.
    • Se não estiver vazia, busca recursivamente o local adequado para inserção.
  • Regra 2: Nó com apenas uma chave (nó simples)
    • Insere o valor diretamente no nó, transformando-o em um nó com duas chaves.
    • Não há necessidade de rebalanceamento.
  • Regra 3: Nó com duas chaves (nó cheio)
    • O valor é inserido e o nó é quebrado (split).
    • A chave mediana é promovida para o nível superior (pai).
    • Caso o pai também tenha duas chaves, a operação de quebra é repetida.
    • Se a quebra alcançar a raiz, uma nova raiz é criada, aumentando a altura da árvore.
No *insere(No **no, int valor, int *rval) {
    No *aux;
    int vaux, promov;

    if (*no == NULL) {
        // Caso base: árvore (ou subárvore) vazia → cria um novo nó folha com uma chave
        *no = criaNo(valor, 0, 1, NULL, NULL, NULL);
        return NULL; // nada a promover
    }

    if (ehFolha(*no)) {
        if ((*no)->num_chaves == 1) {
            // Inserção direta no nó com uma única chave
            adicionaChave(*no, valor, NULL);
            return NULL;
        } else {
            // Nó já cheio → precisa quebrar
            aux = quebraNo(*no, valor, &vaux, NULL);
            *rval = vaux; // chave a ser promovida
            return aux;   // novo nó a ser ligado ao pai
        }
    } else {
        // Nó interno → continua busca recursiva
        if (valor < (*no)->chave_e)
            aux = insere(&((*no)->esq), valor, &vaux);
        else if ((*no)->num_chaves == 1 || valor < (*no)->chave_d)
            aux = insere(&((*no)->centro), valor, &vaux);
        else
            aux = insere(&((*no)->dir), valor, &vaux);

        if (aux == NULL) {
            // Nenhuma quebra ocorreu nas subárvores
            return NULL;
        } else if ((*no)->num_chaves == 1) {
            // Insere a chave promovida no nó atual
            adicionaChave(*no, vaux, aux);
            return NULL;
        } else {
            // Nó cheio → precisa quebrar e promover novamente
            No *novo = quebraNo(*no, vaux, &promov, aux);
            *rval = promov;
            return novo;
        }
    }
}

void insereArvore23(No **raiz, int valor) {
    int promovida;
    No *novo = insere(raiz, valor, &promovida);
    if (novo != NULL) {
        // Quebra da raiz → cria nova raiz com a chave promovida
        *raiz = criaNo(promovida, 0, 1, *raiz, novo, NULL);
    }
}

Remoção em Árvore 2-3

A remoção em árvores 2-3 é feita de forma que a estrutura continue balanceada e todas as folhas permaneçam no mesmo nível.

Etapas da Remoção

  1. Localizar a chave a ser removida
    • Percorre a árvore até encontrar o nó que contém a chave.
  1. Verificar se o nó é folha
    • Se não for folha, a chave é substituída pelo seu sucessor (menor da subárvore à direita) e a remoção é feita na folha que contém o sucessor.
    • Se for folha com duas chaves, remove diretamente a chave desejada.
    • Se for folha com uma única chave, a chave é removida e o nó fica vazio, exigindo reequilíbrio.
  1. Reequilíbrio
    • Necessário quando um nó fica com zero chaves após a remoção.
    • Duas estratégias podem ser aplicadas:

    a) Fusão (Merge):

    • Se o nó e seu irmão têm apenas uma chave, eles são fundidos em um único nó com duas chaves.

    b) Rotação (Redistribuição):

    • Se o irmão do nó tem duas chaves, uma rotação é feita para redistribuir as chaves entre os nós e manter o equilíbrio.

Função de reequilíbrio

int reequilibrar(No **pai, No **filho) {
    No *p = *pai;
    No *f = *filho;

    if (f->num_chaves > 0) return 1;  // Nada a fazer

    // Caso o filho esteja vazio → precisa reequilibrar
    if (f == p->esq) {
        if (p->centro->num_chaves == 2) {
            // Rotação da esquerda para a direita
            f->chave_e = p->chave_e;
            f->num_chaves = 1;
            p->chave_e = p->centro->chave_e;
            p->centro->chave_e = p->centro->chave_d;
            p->centro->num_chaves = 1;
        } else {
            // Fusão com o centro
            f->chave_e = p->chave_e;
            f->chave_d = p->centro->chave_e;
            f->num_chaves = 2;
            free(p->centro);
            p->centro = NULL;
            p->chave_e = p->chave_d;
            p->num_chaves--;
        }
        return 1;
    }

    if (f == p->centro) {
        if (p->esq->num_chaves == 2) {
            // Rotação da direita para a esquerda
            f->chave_e = p->chave_e;
            f->num_chaves = 1;
            p->chave_e = p->esq->chave_d;
            p->esq->num_chaves = 1;
        } else if (p->num_chaves == 2 && p->dir && p->dir->num_chaves == 2) {
            // Rotação da direita
            f->chave_e = p->chave_d;
            f->num_chaves = 1;
            p->chave_d = p->dir->chave_e;
            p->dir->chave_e = p->dir->chave_d;
            p->dir->num_chaves = 1;
        } else {
            // Fusão com a esquerda
            p->esq->chave_d = p->chave_e;
            p->esq->num_chaves = 2;
            free(f);
            p->centro = NULL;
            p->chave_e = p->chave_d;
            p->num_chaves--;
        }
        return 1;
    }

    if (f == p->dir) {
        if (p->centro->num_chaves == 2) {
            // Rotação da esquerda
            f->chave_e = p->chave_d;
            f->num_chaves = 1;
            p->chave_d = p->centro->chave_d;
            p->centro->num_chaves = 1;
        } else {
            // Fusão com o centro
            p->centro->chave_d = p->chave_d;
            p->centro->num_chaves = 2;
            free(f);
            p->dir = NULL;
            p->num_chaves--;
        }
        return 1;
    }

    return 1;
}

Função recursiva de remoção

int removeRecursivo(No **no, int valor) {
    if (*no == NULL) return 0;

    No *n = *no;

    // Caso: nó folha
    if (ehFolha(n)) {
        if (n->num_chaves == 2) {
            if (valor == n->chave_e) {
                n->chave_e = n->chave_d;
                n->num_chaves = 1;
                return 1;
            } else if (valor == n->chave_d) {
                n->num_chaves = 1;
                return 1;
            }
            return 0;
        } else if (n->num_chaves == 1) {
            if (valor == n->chave_e) {
                n->num_chaves = 0;  // marca para reequilibrar
                return 1;
            }
            return 0;
        } else return 0;
    }

    // Caso: chave está em nó interno
    if ((n->num_chaves == 2 && valor == n->chave_d) || valor == n->chave_e) {
        // Substitui pelo sucessor
        No **sub = (valor == n->chave_e) ? &(n->centro) : &(n->dir);
        No *s = *sub;
        while (!ehFolha(s)) s = s->esq;
        int succ = s->chave_e;
        if (valor == n->chave_e) n->chave_e = succ;
        else n->chave_d = succ;
        removeRecursivo(sub, succ);
        return 1;
    }

    // Descida recursiva
    No **child;
    if (valor < n->chave_e)
        child = &(n->esq);
    else if (n->num_chaves == 1 || valor < n->chave_d)
        child = &(n->centro);
    else
        child = &(n->dir);

    int ok = removeRecursivo(child, valor);
    if (!ok) return 0;

    // Reequilibra se necessário
    return reequilibrar(no, child);
}

Função principal de remoção

void removeArvore23(No **raiz, int valor) {
    removeRecursivo(raiz, valor);

    // Se a raiz ficou com 0 chaves e tem filho único, sobe o filho
    if (*raiz && (*raiz)->num_chaves == 0) {
        No *tmp = (*raiz)->esq;
        free(*raiz);
        *raiz = tmp;
    }
}
