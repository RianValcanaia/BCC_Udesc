Na pasta "./pilha" temos o código de uma pilha implementada sobre um vetor.

Na pastas "./aplicacao_A" temos a aplicação "torres de hanoi" que pode ser usada para testar a pilha. 
Existem mais duas alternativas de aplicações para testes:Ocontidas em "./aplicacao_B" e "./aplicacao_C".

Para rodar o exemplo de solução do problema das Torres de Hanoi usando a pilha (PE): crie um diretório 
temporário e para copie os arquivos "./pilha/pilha.c", "./aplicacao_A/hanoi.c" e "./aplicacao_A/arq.h", 
compile os arquivos neste neste diretório temporário.

O memso pode ser feito para rodr as outras duas alternativas de aplicações.

Você pode criar um Makefile para automatizar esse processo.
